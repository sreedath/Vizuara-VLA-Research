Our method monitors the hidden-state embeddings of a Vision-Language-Action (VLA) model to detect out-of-distribution visual inputs at near-zero computational cost.

Pipeline Overview:
1. CALIBRATION PHASE (offline, one-time):
   - Collect a small set of 2-8 clean driving scene images
   - Run each through the VLA model (OpenVLA-7B), extracting the hidden state embedding at layer 3 (last token position) — a 4096-dimensional vector
   - Compute the centroid (mean) of these embeddings to form the reference point
   - Compute a 3-sigma threshold from the calibration distances

2. DETECTION PHASE (online, per-frame):
   - For each new input image, run the standard VLA forward pass
   - Extract the hidden state at layer 3 (last token position) — no extra computation needed
   - Compute the cosine distance from the precomputed centroid
   - If cosine distance exceeds the threshold, flag the input as out-of-distribution

Key insight: The VLA's OUTPUT confidence is completely uninformative (assigns identical confidence 0.56 to clean driving scenes and random noise, p=0.321). But the INTERNAL hidden-state embeddings contain a perfect OOD signal because: (a) the model is deterministic in eval mode (zero within-image variance), (b) even small corruptions shift the embedding by orders of magnitude above the noise floor.

Result: AUROC = 1.0 across fog, night, noise, and blur corruptions, with only 14.6 microseconds of additional computation (0.01% overhead).
